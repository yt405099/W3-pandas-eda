# W3-pandas-eda
WHO Life Expectancy Exploratory Data Analysis (EDA)

## Objective
The objective of this project is to perform an exploratory data analysis (EDA)
on publicly available WHO life expectancy data to understand global patterns,
regional differences, and temporal trends related to life expectancy.

## Dataset
- Source: Kaggle – Life Expectancy (WHO)
- Description: The dataset contains country-level life expectancy data along with
  health, economic, and demographic indicators across multiple years.

## Analysis Overview
The analysis includes:
- Initial data inspection and structure verification
- Missing value analysis
- Descriptive statistics
- Distribution analysis of key variables
- Correlation analysis
- Country and regional comparisons
- Time-series analysis of life expectancy trends
